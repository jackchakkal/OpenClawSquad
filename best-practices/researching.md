# Pesquisando — Melhores Práticas

## Princípios Fundamentais

1. **Verificar fontes.** Nunca use uma única fonte. Mínimo de 3 fontes diferentes para утверд information.

2. **Distinguir fato de opinião.** Separe claramente dados objetivos de interpretações.

3. **Citar sempre.** Cada reivindicação factual deve ter fonte.

4. **Atualização.** Verifique a data das fontes. Informações desatualizadas podem ser perigosas.

5. **Múltiplos ângulos.** Para temas controversos, apresente múltiplas perspectivas.

## Processo de Pesquisa

### 1. Definição do Escopo
- Definir perguntas de pesquisa claras
- Identificar fontes primárias e secundárias
- Estabelecer critérios de inclusão/exclusão

### 2. Coleta
- Buscar em múltiplas fontes
- Documentar metadados (data, autor, URL)
- Organizar por tema/relevância

### 3. Análise
- Identificar padrões
- Verificar contradições
- Sintetizar insights

### 4. Apresentação
- Resumir achados principais
- Destacar lacunas de conhecimento
- Sugerir próximos passos

## Fontes Confiáveis

### Excelentes
- Estudos acadêmicos revisados por pares
- Relatórios de organizações reconhecidas
- Dados governamentais oficiais
- Fontes jornalísticas estabelecidas

### Evitar
- Redes sociais como fonte primária
- Sites sem autor identificado
- Informações sem data
- Claims sem evidência

## Formato de Brief de Pesquisa

```markdown
# Título da Pesquisa

## Perguntas de Pesquisa
1. [pergunta 1]
2. [pergunta 2]

## Fontes
### Fonte 1
- Título: ...
- URL: ...
- Data: ...
- Relevância: ...

## Achados
### Ponto 1
- Descrição: ...
- Fonte: ...

## Síntese
[Resumo dos achados]

## Limitações
[O que não foi coberto]

## Próximos Passos
[Sugestões]
```
