# YouTube Script — Melhores Práticas

## Estrutura do Vídeo

### 1. Hook (0:00-0:30)
- Parar o scroll
- Prometer valor
- Dados ou pergunta forte

### 2. Introdução (0:30-1:00)
- Quem é você
- O que vai aprender
- Por que isso importa

### 3. Corpo
- Dividir em seções
- Cada seção = 2-5 minutos
- Transições entre seções

### 4. Conclusão (últimos 2-3 minutos)
- Resumo
- Próximo passo
- CTA para inscrição

## Estrutura do Roteiro

### Timestamp
- Marca d'água
- O que será coberto

### Script
- Texto completo
-インicações de tom
- Notes para edição

### B-Roll
- Sugestões de filmagem
- Screenshots
- Imagens de referência
