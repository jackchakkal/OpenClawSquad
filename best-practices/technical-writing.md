# Technical Writing — Melhores Práticas

## Estrutura

### Introdução
- O que será coberto
- Pré-requisitos
- Por que isso é importante

### Corpo
- Seções claras
- Passo a passo
- Exemplos de código
- Screenshots quando relevante

### Conclusão
- Resumo
- Próximos passos
- Recursos adicionais

## Escrita Técnica

### Clareza
- Frases curtas
- Linguagem simples
- Evitar jargão desnecessário

### Precisão
- Dados corretos
- Comandos exatos
- Versões específicas

### Estrutura
- Subtítulos claros
- Listas para passos
- Tabelas para dados
- Código em blocos

## Formatação

### Código
- Syntax highlighting
- Comentários explicativos
- Copy-paste ready

### Imagens
- Screenshots claros
- Anotações quando necessário
- Alt text descritivo

### Tabelas
- Dados tabulados
- Cabeçalhos claros
- Fontes quando aplicável

## Revisã

### Checklist
- [ ] Instruções funcionam
- [ ] Código testado
- [ ] Links válidos
- [ ] Imagens carregam
- [ ] grammar/spelling

### Leitor Beta
- Alguém sem conhecimento prévio
- Testar os passos
- Identificar lacunas
