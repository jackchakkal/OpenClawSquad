# Blog Post — Melhores Práticas

## Estrutura do Artigo

### Título
- Keyword principal
- Promessa clara
- Números quando relevante

### Meta Description
- 150-160 caracteres
- Resumo atrativo
- Incluir CTA

### Introdução
- Hook (pergunta, dado, história)
- Contexto
- O que será coberto

### Corpo
- Subtítulos claros (H2, H3)
- Parágrafos curtos
- Listas quando possível
- Imagens para ilustrar

### Conclusão
- Resumo dos pontos
- Call to action
- Próximo passo

## Tipos de Artigo

### 1. How-to
- Problema
- Solução passo a passo
- Resultado esperado
- Dicas extras

### 2. List Post
- Introdução
- Lista numerada
- Cada item com explicação
- Conclusão

### 3. Contrário/Opinião
- Posição clara
- Argumentos
- Evidência
- Conclusão

### 4. Comparativo
- Introdução
- Opção A
- Opção B
- Comparação
- Veredicto

### 5. Case Study
- Situação inicial
- Desafio
- Solução
- Resultado

## SEO On-Page

### Keywords
- Título principal
- Primeiro parágrafo
- Subtítulos
- Ao longo do texto (natural)
- URL

### Links
- Internos para相关内容
- Externos para autoridades
- Âncora otimizada

### Imagens
- Alt text descritivo
- Nome do arquivo otimizado
- Comprimir para velocidade

## Engajamento

### CTA
- Dentro do artigo
- Final do artigo
- Sidebar (se houver)

### Interação
- Perguntas
- Comentários incentivados
- Social share
