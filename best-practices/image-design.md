# Image Design — Melhores Práticas

## Princípios Visuais

### Hierarquia
- Tamanho: o maior = mais importante
- Cor: contraste chama atenção
- Posição: canto superior esquerdo

### Equilíbrio
- Visual e textual
- Espaçamento
- Alinhamento

### Cores
- Paleta: 3-5 cores
- Contraste: texto legível
- Consistência de marca

### Tipografia
- Máximo 2 fontes
- Tamanho legível
- Hierarquia clara

## Dimensões por Plataforma

### Instagram
- Feed: 1080x1080 ou 1080x1350
- Story: 1080x1920
- Carrossel: 1080x1350

### LinkedIn
- Post: 1200x627
- Article: 1128x191

### Twitter/X
- Post: 1200x675
- Header: 1500x500

### YouTube
- Thumbnail: 1280x720
- Banner: 2560x1440

## Elementos de Design

### Hook Visual
- Cor contrastante
- Texto grande
- Elemento surpreendente

### CTA
- Botão claro
- Cor que destaca
- Posição estratégica

## Ferramentas

### HTML/CSS Rendering
- Self-contained HTML
- Inline CSS
- Web-safe fonts
- Testar viewport

### Geração de Imagem
- DALL-E
- Midjourney
- Canva
- Stable Diffusion

## Checklist

- [ ] Dimensões corretas
- [ ] Texto legível
- [ ] Cores de marca
- [ ] CTA claro
- [ ] Visual hook
- [ ] Mobile-friendly
