# Twitter/X Thread — Melhores Práticas

## Estrutura da Thread

### Tweet 1: Hook (CRÍTICO)
- Parar o scroll
- Prometer valor
- Criar curiosidade

### Tweets 2-X: Corpo
- Desenvolver o tema
- Uma ideia por tweet
- Dados e exemplos

### Último Tweet: CTA
- Resumir ponto principal
- Pedir engajamento
- Prometer mais conteúdo

## Tipos de Thread

### 1. Tutorial/How-to
1. Hook: Resultado que alguém quer
2. Explicação passo a passo
3. Cada tweet = 1 passo
4. Exemplo no final
5.CTA: Save/Share

### 2. Storytelling
1. Hook: Situação inicial
2. Desenvolvimento: Desafio
3. Clímax: Ponto de virada
4. Resolução: Lição
5. CTA: Reflexão

### 3. Lista
1. Hook: "X coisas que..."
2. X tweets = X itens
3. Explicação de cada um
4. Resumo
5. CTA

### 4. Contrário
1. Hook: Opinião forte
2. Contexto/Evidence
3. Desenvolver argumento
4. Chamada para pensar
5. CTA

## Melhores Práticas

### Tweet Inicial
- Use emojis estratégicos
- Número de tweets no hook: "🧵 7 ways to..."
- Pergunta provocadora

### Corpo
- Número cada tweet: "2/" até "N/"
- Imagens aumentam engajamento
- mantenha cada tweet conciso

### Engajamento
- Responda comentários na thread
- RTC (Reply Triangular) aumenta alcance
- Pergunte ao final

### Timing
- Poste threads entre 8-10h ou 17-19h
- Segunda-quarta: melhor
- Responda rápido após postar

## Formatação

### Thread Card
- Ative no Twitter: "Show thread"
- Título claro
- Imagem de capa opcional

### Imagens
- Use para ilustrar pontos
- Screenshots de ferramentas
- Gráficos simples
