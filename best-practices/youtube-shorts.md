# YouTube Shorts — Melhores Práticas

## Especificações Técnicas

### Dimensões
- Resolução: 1080 x 1920 (9:16)
- Duração: até 60 segundos (ideal: 30-45s)

### Formato
- Vídeo vertical
- Áudio claro
- Legendas essenciais

## Estrutura do Short

### 0-3 Segundos: Hook
- Visual chocante ou surpreendente
- Pergunta forte
- "Parar o scroll" imediato

### 3-30 Segundos: Valor
- Uma ideia central
- Explicação rápida
- Exemplo concreto

### 30-45 Segundos: CTA
- "Se inscreva para mais"
- Pergunta para comentários

## Tipos de Shorts que Viralizam

### 1. Dica Rápida
- "Uma dica que..."
- Resultado imediato
- Exemplo

### 2. Fato Interessante
- Dado surpreendente
- Contexto mínimo
- Reação

### 3. Contrário
- "Todo mundo está errado sobre..."
- Sua perspectiva

### 4. Storytime
- Situação rápida
- Ponto-chave
- Lição

### 5. Tutorial Rápido
- Passo 1-2-3
- Resultado

## Hashtags

### Estratégia
- #Shorts #Shorts #viral (obrigatório)

- Centrale

### Prompts

- Perguntas simples
