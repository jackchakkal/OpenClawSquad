# 🎼 OpenClawSquad

**Sistema de Orquestração Multi-Agente do Carinha**

---

## O que é?

O OpenClawSquad é meu sistema de squads de IA — equipes de agentes especializados que trabalham juntos para完成 tarefas complexas. Sou eu (Carinha) coordenando tudo! 😄

---

## Agentes

| Agente | Ícone | Papel |
|--------|-------|-------|
| **Coordinator** | 🎼 | Eu! Coordeno tudo e garanto que funcione |
| **Researcher** | 🔍 | Pesquisa e coleta informações |
| **Strategist** | 🎯 | Analisa dados e cria estratégias |
| **Writer** | ✍️ | Produz conteúdo written |
| **Executor** | ⚡ | Executa ações e comandos |
| **Reviewer** | 🔎 | Garante qualidade |

---

## Squads Prontos

### 🔒 Pentest
Teste de invasão e segurança
```
Coordinator → Researcher → Executor → Reviewer → Coordinator
```

### 📝 Content
Criação de conteúdo
```
Coordinator → Researcher → Writer → Reviewer → Coordinator
```

### 🔬 Research
Pesquisa profunda
```
Coordinator → Researcher → Strategist → Coordinator
```

### ⚙️ Automation
Automação de tarefas
```
Coordinator → Strategist → Executor → Reviewer → Coordinator
```

---

## Como Usar

### Modo 1: Eu escolho
> "Faça uma pesquisa sobre X"
→ Eu decido qual squad usar

### Modo 2: Squad específico
> "Use o squad de pentest no site X"

### Modo 3: Múltiplos agentes
> "Chame o Researcher e o Writer"

### Modo 4: Com checkpoint
> "Faça X, mas pause para eu aprovar"

---

## Checkpoints

Paro para você approve em momentos importantes:
- ✅ Antes de executar ações externas
- ✅ Após coletar informações
- ✅ Antes de entregar o resultado
- ✅ Quando há múltiplas opções
