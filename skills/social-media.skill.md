# Skill: Social Media

Cria conteúdo otimizado para redes sociais.

- Texto curto e impactante

### Carrosséis
- Estrutura de 5-10 slides
- Fluxo narrativo
- Design consistente

### Estratégia
- Timing de postagem
- Engajamento
- Crescimento de audiência
