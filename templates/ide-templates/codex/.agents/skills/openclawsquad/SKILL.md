# AGENTS.md - OpenClawSquad for Codex

This file configures Codex to work with OpenClawSquad squads.

## Quick Commands

- `squad:create <name>` - Create a new squad
- `squad:run <name>` - Run a squad pipeline
- `squad:list` - List available squads

## Usage

Create squads in the `squads/` directory, then run them with the OpenClawSquad CLI.
