# AGENTS.md - OpenClawSquad

This workspace uses OpenClawSquad for multi-agent orchestration.

## About

OpenClawSquad is an open-source alternative to opensquad with:
- Real-time WebSocket dashboard
- 27+ pre-built agents
- IDE integrations (Claude Code, Codex, Cursor, Windsurf, Zed, Jan)
- Dynamic squad creation

## Commands

```bash
# Create a new squad
npx openclawsquad create my-team

# Run a squad
npx openclawsquad run my-team

# Start dashboard
npx openclawsquad dashboard
```

## Squads

Squads are defined in `squads/` directory.
